Integritetspolicy

Teknikbase värnar om din integritet. Den här sidan beskriver i korthet hur uppgifter kan hanteras när du använder webbplatsen.

Kontaktformulär

När du skickar ett meddelande via kontaktformuläret behandlas de uppgifter du lämnar, till exempel namn, e-postadress och meddelandeinnehåll, för att kunna besvara din förfrågan.

Affiliatelänkar

Teknikbase kan använda affiliatelänkar. Det innebär att externa samarbetspartners kan använda spårning för att registrera klick och köp som sker via länkar från webbplatsen.

Cookies och statistik

Webbplatsen kan använda grundläggande teknik för att förbättra funktion, säkerhet och statistik. Om ytterligare analysverktyg läggs till kan policyn uppdateras.

Dina frågor

Har du frågor om integritet eller databehandling är du välkommen att kontakta oss via kontaktsidan.
